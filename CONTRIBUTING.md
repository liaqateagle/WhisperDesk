# Contributing to WhisperDesk

Thank you for your interest in contributing to WhisperDesk! We welcome contributions of all kinds.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/whisperdesk.git`
3. Install dependencies: `npm install`
4. Run the dev server: `npm run tauri dev`

## Prerequisites

- **Node.js** 18+
- **Rust** 1.70+
- **Tauri v2 prerequisites** — see [Tauri docs](https://v2.tauri.app/start/prerequisites/)

## Development

```bash
# Install frontend dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```

## Reporting Issues

Please report bugs and request features on [GitHub Issues](https://github.com/liaqateagle/whisperdesk/issues).
